Bổ sung thêm: Quick Actions (4 nút tắt: lịch sử, thành tựu, xếp hạng, trợ giúp), avatar người dùng tròn trong Rank Hero, points badge dạng pill với icon SVG riêng mỗi loại.

ProfileScreen.kt + RankHero + QuickActions + UnlockList + Ledger SVG icons

  • Signal / Battery — status bar
  • Back chevron — header trái
  • Gear / Settings — header phải, 8 tia
  • User avatar — silhouette trong vòng tròn
  • Bamboo deco — rank hero background
  • Karma (Nghiệp) — yin-yang đơn giản
  • Flame (Streak) — ngọn lửa
  • Lightning (Energy) — tia sét
  • Crystal eye (AI) — mắt thần / pha lê
  • Fortune card — thẻ bài ngôi sao
  • Check-circle — đã mở khoá
  • Quick: clock, star, trophy, question
  • Ledger: flame, checkbox, card, orb
  • Bottom nav: calendar, pencil, book, grid
